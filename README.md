## How to build & run

```bash
npm i
npm run build
npm start
```

Open [localhost:8001](http://localhost:8001/)

## For development

```bash
npm run watch
```

Open [localhost:3000](http://localhost:3000/)

## Proxy mode

```bash
npm run proxy
```

Both ports `3000` and `8001` are available.
